# ModularServer

A modular, command-line server built in C# that helps developers create networked applications without deep networking knowledge.

It supports:
- TCP / UDP
- Lua scripting
- Built-in database
- Secure client-server communication

The goal is to make building online features simpler and more flexible for new programmers.

⚠️ This project was discontinued before being fully completed or polished.
